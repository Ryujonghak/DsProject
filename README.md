추가사항) 
  - 현재 백엔드의 경우 Oracle Cloud를 이용하여 DB를 구축하였기 때문에 깃허브 레퍼지토리의 파일로는 정상작동이 되지 않습니다.
  - 프론트의 경우 계속해서 새로운 날짜로 검색 및 업데이트가 이루어집니다. 백엔드에 현재 날짜를 기점으로 영화 스케쥴및 데이터를 업로드해야 정상작동이 가능합니다.
